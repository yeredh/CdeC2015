## Paquetes de Bioconductor Los siguientes comandos son para instalar los paquetes de Bioconductor que vamos a usar el resto de la clase. Copia el codigo y ejecutalo en tu consola. ``` # Instalar el paquete Biobase source("http://bioconductor.org/biocLite.R") biocLite() # Instalar paquetes de Bioconductor biocLite(c("affy","oligo","GEOquery","rae230a.db", "AnnotationDbi","biomaRt","ALL","genefilter", "multtest","hgu95av2.db","GSEABase","KEGG.db")) ``` * Necesitas una conexion a internet para poder instalar estos paquetes * Puede ser algo tardado.
